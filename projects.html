Smart Exam Hall

My Contribution

Front-end developer + interfacing both front-end and back-end.

Description

System for exam halls to automate some of its operational tasks.

Features

● Fingerprint scanner to register, verify and mark the attendance of students.
● Display module to view the fingerprint scanning status and allocated computer in the exam hall.
● Client side desktop app to display student information after computer allocation and remaining time for examination.
● Backend to randomly allocate students for the computers and to handle front-end processes.
● Web app to perform administrative operations.

Key technologies

● Front-end: Flutter, React.js
● Back-end: Node.js, REST APIs
● Database: SQL

Hardware technologies

● Microcontroller: ESP32
● Others: FPM10A Fingerprint module, Custom wireless network
